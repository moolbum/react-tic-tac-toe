## 🖥 프로젝트 실행 방법

1. git clone
   ```bash
   git clone https://github.com/moolbum/react-tic-tac-toe.git
   ```
2. 패키지 설치
   ```bash
   yarn install
   ```
3. local 프로젝트 실행
   ```bash
   yarn start
   ```
